# Weibo Hot Search Crawler & Data Analysis (微博热搜爬取与数据分析)

这是一个基于 Python 的微博热搜数据采集、解析、清洗与数据分析项目。项目模块化设计，适合学习数据采集（爬虫）以及基础的数据分析与清洗流程。

## 📌 项目结构

项目由多个独立的脚本组成，按顺序执行可以完成从数据采集到数据清洗分析的完整流程：

*   **`common.py`**：公共配置文件，包含微博热搜接口地址及伪装浏览器的请求头（Headers）。
*   **`1_fetch_data.py`**：数据采集模块，负责从微博 API 获取热搜原始数据。
*   **`2_parse_and_save.py`**：数据解析与存储模块，解析原始 JSON 数据并将其保存为 CSV 文件（`weibo_hot_search.csv`）。
*   **`3_data_shape.py`**：查看数据集的维度（行数与列数）。
*   **`4_data_stats.py`**：进行基础的描述性统计分析（如均值、最大值等）。
*   **`5_data_preview.py`**：预览数据集前几行的内容。
*   **`6_missing_values.py`**：检测数据集中的缺失值情况。
*   **`7_duplicate_values.py`**：检测数据集中的重复记录。
*   **`8_heat_distribution.py`**：分析并展示热搜热度值的分布情况。
*   **`9_tag_distribution.py`**：分析热搜标签（如“新”、“热”、“爆”等）的分布情况。
*   **`10_data_cleaning.py`**：数据清洗模块，处理缺失值、重复记录与异常热度值，并将清洗后的数据保存为 `weibo_hot_search_cleaned.csv`。

## 🚀 快速开始

### 1. 环境准备
确保已安装 Python 3.x 以及以下第三方依赖库：

```bash
pip install requests pandas
```

### 2. 采集并保存热搜数据
运行以下命令获取最新的微博热搜数据并保存为本地 CSV 文件：

```bash
python 2_parse_and_save.py
```

### 3. 执行数据清洗
运行清洗脚本，处理数据中的异常、缺失和重复项：

```bash
python 10_data_cleaning.py
```

### 4. 进行数据分析
你可以运行 `3_data_shape.py` 到 `9_tag_distribution.py` 中任意脚本来探索和分析采集到的热搜数据。

## 📊 数据字段说明

导出的 CSV 文件包含以下字段：

| 字段名 | 说明 |
| :--- | :--- |
| **排名** | 热搜榜单的实时排名 |
| **话题词** | 热搜话题的关键词/文本内容 |
| **热度值** | 微博官方的热度指数数值 |
| **标签** | 话题标记（如：新、热、爆、商、无 等） |
| **采集时间** | 获取该条热搜的具体系统时间 |
